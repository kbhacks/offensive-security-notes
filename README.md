# Offensive Security Notes

Personal knowledge base for penetration testing, red teaming, and CTF challenges.

## Structure

- `writeups/` — Machine and challenge writeups (HTB, THM, CTF)
- `active-directory/` — AD enumeration, attacks, lateral movement, persistence
- `web-security/` — OWASP Top 10 exploitation notes and methodology
- `privilege-escalation/` — Linux and Windows privesc techniques
- `networking/` — Protocol notes and cheatsheets
- `methodology/` — Recon, enumeration, exploitation, and reporting frameworks

## Disclaimer

All content is for authorized security testing and educational purposes only. No real target data is included.
